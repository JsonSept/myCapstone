API Routes and Methods

Home

http Methods Path Description
GET / Takes you to the homepage

Users

GET /users Displays all the users with their info
GET /users/:id Displays a single user with their info
POST /users/register Adds a new user
PATCH/PUT /users/update/:id allows you to edit a users info
DELETE /users/delete/:id allows you to delete a users info

Products/items

GET /products Displays all the products with their info
GET /products/:id Displays a single user with their info
POST /products/register Adds a new user
PATCH/PUT /products/update/:id allows you to edit a products info
